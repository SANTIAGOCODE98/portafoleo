J'ai eu l'occasion de réaliser des projets de programmation à des fins éducatives, utilisant des langages tels que Python, Java et JavaScript pour leur mise en œuvre. Ces projets avaient divers objectifs, incluant le développement frontend, le développement backend et l'analyse de données

J'ai de l'expérience dans l'utilisation de bases de données pour la gestion des projets que j'ai réalisés. J'ai travaillé avec des langages SQL et NoSQL, utilisant MariaDB ou MySQL pour les bases de données relationnelles, et MongoDB pour les bases de données non relationnelles

Je possède des connaissances sur le fonctionnement du cloud et des services qui y sont déployés, en utilisant principalement AWS comme base. J'ai de l'expérience dans la gestion de conteneurs avec Docker pour assurer le déploiement et le bon fonctionnement des projets sur plusieurs versions.
